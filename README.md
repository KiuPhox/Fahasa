## Installing / Getting started
1. Install [XAMPP](https://www.apachefriends.org/download.html)
2. Import database.sql in database folder
3. Install all required packages
```
composer install
```
4 Setup .env
```shell
DB_HOST=localhost
DB_USER=root
DB_PASS=
DB_DATABASE=Fahasa
PHPMAILER_USER=example@gmail.com
PHPMAILER_PASS=pass
```
